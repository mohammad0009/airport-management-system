# Airport Management System

A relational database system for managing airport operations, built as a university project for Database design & development.

## What it includes
- 7 tables: Passenger, Employee, Flight, Plane, Department, Passenger_Flight, Passenger_PhoneNumber
- 4 views: affordable flights, big planes, passenger info, department salaries
- 5 stored procedures for querying employees, flights, and passenger data
- Role-based security: 3 users (admin, employee, passenger) with different privileges
- PHP web interface for passengers and employees

## Tech Used
MySQL · MariaDB · PHP · phpMyAdmin

## Files
- `final.sql` — full database schema, data, views, and constraints
- Documentation — full technical report including schema design, testing, and security
